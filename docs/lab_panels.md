# Lab Panels (What / Why / When)

| Panel/Test | Why it matters here | When |
|---|---|---|
| **Histamine / Tryptase** | Objective marker for mast-cell activation during/after flares | During flare or within 2–4h |
| **CRP (±ESR)** | Inflammatory burden | Baseline + during flare |
| **CMP, CBC** | Electrolytes, liver/kidney, anemia | Baseline; med changes |
| **Fasting glucose, HbA1c** | Metabolic status | Baseline + q3–6mo |
| **Cortisol (AM/PM)** | HPA axis rhythm | Baseline; symptomatic |
| **Testosterone (total/free), Estradiol** | Endocrine balance on T/AI | Baseline; dose changes |
| **B12, Folate, Homocysteine** | Methylation profile | Baseline; supplement changes |
| **Vit D, Ferritin/iron studies** | Fatigue, immune tone | Baseline; replete if low |
